HTML Table Reference

Table Tags

Tag Name Description
form form représente un formulaire
input input est utilisé pour créer un contrôle interactif dans un formulaire web qui permet l'utilisateur de saisir des données.
label label représente une légende pour un objet d'une interface utilisateur.
radio radio sont généralement utilisés pour construire des groupes d'options parmi lesquelles on ne peut choisir qu'une valeur.
button button représente un bouton cliquable, utilisé pour soumettre des formulaires
textarea textarea représente un bouton cliquable, utilisé pour soumettre des formulaires
checkbox checkbox eprésente un contrôle qui permet d'éditer du texte sur plusieurs lignes
